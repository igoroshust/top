## 47 Сегментирование рынка. Основания для сегментирования покупательских и промышленных товаров. Выбор целевых сегментов: недифференцированный, дифференцированный и концентрированный маркетинг.

Сегментирование рынка — это процесс разделения общего рынка на отдельные группы (сегменты) потребителей, которые имеют схожие характеристики, потребности и поведение. Цель сегментации — повысить эффективность маркетинга за счёт адаптации продуктов, услуг и маркетинговых стратегий под конкретные группы, что способствует увеличению продаж и лояльности клиентов


### Основания для сегментирования

**Для покупательских (потребительских) товаров используются следующие критерии:**
- Географический. Разделение рынка по регионам, городам, климату, плотности населения. Например, больше тёплой одежды для северных регионов. 
- Демографический. Учёт возраста, пола, дохода, образования, семейного положения. Например, продукты для молодых семей или спортивная одежда для молодёжи. 
- Психографический. Анализ личностных качеств, ценностей, стиля жизни, интересов. Например, продукты для вегетарианцев или эко-активистов. 
- Поведенческий. Изучение поведения потребителей, использования продукта, степени лояльности. Например, программы лояльности для постоянных клиентов. 
- Социальный. Учёт социального статуса, профессиональной деятельности. Например, люксовые товары для состоятельных людей или доступные решения для студентов

**Для промышленных (B2B) товаров применяются другие критерии:**
- По группам продуктов. Учитываются запросы и предпочтения покупателей, касающиеся качественных характеристик продукта. Основные признаки — функциональные и технические показатели, стоимость и т. п.. 

- По предприятиям (конкурентам). Группирование фирм-соперников по параметру конкурентоспособности при продвижении на рынок. Базовые признаки — качественные показатели продукта, стоимость, каналы сбыта, продвижение товара. 

- Демографический. Учёт отрасли, размера компании, местонахождения. Это помогает определить, какие отрасли промышленности обслуживать, какие размеры компаний можно покрыть, в каких географических регионах работать. 

- Операционный. Учёт технологий, используемых в производстве, статуса пользователя, объёма закупок. 
- Закупочный. Учёт процедур централизованных закупок, которые проводят компании или органы власти. ы
- Ситуационный. Учёт специфических обстоятельств, влияющих на покупку (например, срочность, сезонность).

### Выбор целевых сегментов
После сегментации рынка компания оценивает привлекательность каждого сегмента и выбирает один или несколько для работы. 

**При оценке учитывают:**
- размер сегмента и скорость его изменения (роста, уменьшения);
- структурную привлекательность сегмента (уровень конкуренции, возможность замены продукта, силу позиций покупателей и поставщиков);
- цели и ресурсы организации. 

**Для выбора целевых сегментов можно использовать модель MASDA:**
- Measurable (измеримость): можно ли посчитать количество людей в сегменте?
- Accessible (доступность): есть ли каналы для коммуникации с сегментом?
- Substantial (существенность): достаточно ли велик сегмент для получения прибыли?
- Differentiable (дифференцируемость): реагируют ли потребители сегмента на предложение иначе, чем другие?
- Actionable (реализуемость): хватит ли у компании ресурсов для работы с сегментом?

### Стратегии охвата рынка
Существует три основных подхода к работе с целевыми сегментами:

1. Недифференцированный (массовый) маркетинг. Компания игнорирует различия между сегментами и выходит на весь рынок с одним продуктом. Фокус делается на общих потребностях потребителей, а не на их различиях. Используются массовые системы товародвижения и рекламные кампании. Это позволяет достичь экономии затрат. Пример — начальный этап развития Coca-Cola, когда всем потребителям предлагался один напиток в бутылке одного размера. 

2. Дифференцированный маркетинг. Компания работает с несколькими сегментами, разрабатывая для каждого из них отдельные продукты и маркетинговые стратегии. Цель — достичь большего объёма продаж и укрепить позиции на каждом сегменте за счёт адаптации предложений. Пример — компания BMW, которая предлагает широкий ассортимент продукции для разных сегментов. 

3. Концентрированный (сфокусированный) маркетинг. Компания сосредотачивает усилия на одном или нескольких узких сегментах (нишах), стремясь занять в них доминирующее положение. Это стратегия специализации, которая позволяет предложить продукт, максимально адаптированный под потребности конкретного сегмента. Пример — компания Apple, чья продукция ориентирована на опытных и требовательных пользователей, интересующихся современными технологиями

| Критерий | Недифференцированный (массовый) маркетинг | Дифференцированный маркетинг | Концентрированный (сфокусированный) маркетинг |
|--------|--------------------------------------------|-----------------------------|-----------------------------------------------|
| **Суть стратегии** | Игнорирование различий между сегментами, выход на весь рынок с одним продуктом | Работа с несколькими сегментами, разработка отдельных продуктов и маркетинговых стратегий для каждого | Сосредоточение усилий на одном или нескольких узких сегментах (нишах) |
| **Целевая аудитория** | Весь рынок без сегментации | Несколько сегментов (по интересам, возрасту, доходу и т. п.) | Один узкий сегмент/ниша |
| **Продукт/предложение** | Стандартный продукт для всех потребителей | Уникальные адаптированные предложения для каждого сегмента | Специализированный продукт, максимально отвечающий потребностям конкретной ниши |
| **Подход к маркетингу** | Массовые рекламные кампании, единые каналы продвижения | Отдельные маркетинговые стратегии для каждого сегмента (разные рекламные сообщения, каналы продвижения) | Целенаправленные маркетинговые усилия в рамках выбранной ниши |
| **Система товародвижения** | Единая для всего рынка | Раздельная для каждого сегмента | Оптимизированная под нужды конкретного сегмента |
| **Преимущества** | - Экономия затрат за счёт масштаба<br>- Простота управления<br>- Массовый охват аудитории | - Увеличение доли рынка<br>- Глубокие отношения с разными группами клиентов<br>- Возможность максимизировать продажи за счёт адаптации | - Сильная позиция в нише<br>- Высокая лояльность целевой аудитории<br>- Возможность установления премиальных цен<br>- Эффективное использование ограниченных ресурсов |
| **Недостатки** | - Риск потери актуальности при появлении новинок<br>- Ограниченная гибкость<br>- Низкая персонализация<br>- Уязвимость перед конкурентами с дифференцированным подходом | - Высокие затраты на разработку и продвижение нескольких продуктов<br>- Риск распыления ресурсов<br>- Сложность управления множеством стратегий | - Ограниченный объём продаж<br>- Уязвимость при изменении рынка или потребностей ниши<br>- Зависимость от одного сегмента |
| **Требования к ресурсам компании** | Умеренные (достаточно для массового производства и продвижения) | Высокие (необходимы ресурсы для разработки, производства и продвижения нескольких продуктов) | Относительно низкие (фокусировка на одном сегменте снижает затраты) |
| **Примеры** | - Coca‑Cola (на начальном этапе — один напиток для всех)<br>- Соль, сахар, базовые продукты питания | - BMW (разные модели для разных сегментов)<br>- Procter & Gamble (разные бренды моющих средств для разных потребностей) | - Tesla (на старте — электромобили для премиум‑сегмента)<br>- Apple (продукция для технически продвинутых пользователей)<br>- Производитель специализированного медицинского оборудования для узкой области медицины |
| **Когда эффективна** | - На однородных рынках<br>- При ограниченном бюджете<br>- Для базовых товаров первой необходимости | - При наличии ресурсов и возможностей для диверсификации<br>- На рынках с чётко выраженными сегментами<br>- В условиях высокой конкуренции | - Для стартапов и малого бизнеса<br>- При наличии уникальной экспертизы в конкретной области<br>- На нишевых рынках с неудовлетворёнными потребностями |

Выбор стратегии зависит от ресурсов компании, особенностей продукта, степени однородности рынка и других факторов
