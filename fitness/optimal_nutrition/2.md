# Индивидуальный расчёт суточных энерготрат

## Расчёт суточных энерготрат
1. `Основной обмен веществ (Харрис-Бенедикт) - 665 + (9.6 x 47) + (1.85 x 160) - 4.68 x 39 = 1220 ккал`
2. СДДП (специфически динамическое действие пищи) = 10%. = 122 ккал (в данном случае СДДП это энергия на усвоение сложной пищи: белков, жиров и углеводов)
3. Ходьба: вес x время x интенсивность
```python
Интенивность: 3 ккал - лёгкая нагрузка (ходьба); 5 ккал - тренировка; 7 ккал - интенсивная сложная тренировка
```
Пример расчёта ходьбы: 47 x 0.5 x 3 = 70 ккал
```python
47 - вес
0.5 - полчаса; 1 - час
3 - активность
```
4. ДЭТ (дополнительные энерготраты, хобби вроде танцев и тд): вес x время x интенивность
47 x 4 x 5 = 940 / 7 = 134 (женщина весом 47кг в течении 4 часа ведёт интенсивную тренировку по танцам)
940 распределяем на 7 дней в неделю

5. Суммируем: Основной обмен веществ + СДДП + Ходьба + ДЭТ
1220 + 122 + 70 + 134 = 1546 ккал - суточные энерготраты в калориях

6. Уточняем цель клиента. Если "похудение" - вычитаем 20%, "набор" - прибавляем 20%.

7. Переводим полученное значение в белки, жиры и углеводы.

8. Составляем рацион (переводим БЖУ в еду)

## Основной обмен
Основной обмен (BMR - Basal Metabolic Rate): это минимальная энергия, необходимая для поддержания жизнедеятельности в состоянии полного покоя (без физической активности, натощак, при комфортной температуре). Расчёт по формуле МФ или ХБ и составляет ~60-70% суточных энерготрат

## Суточные энерготраты
Суточные энерготраты (TDEE - Total Daily Energy Expenditure): это общий расход энергии за день, включая BMR + термогенез активности (NEAT - неупражнительная активность вроде ходьбы или сидячей работы; EAT - упражнения) и термогенез пищи (TEF - энергия на переработку еды, ~10% от калорий). TDEE всегда выше BMR.

### СДДП (другое)
СДДП - суточная доза диетического питания. Это рекомендуемая суточная норма потребления питательных веществ (витаминов, минералов, макронутриентов), необходимая для поддержания здоровья, предотвращения дефицитов и обеспечения нормального функционирования организма.
Учитывает возраст, пол, вес, активность и физиологическое состояние (например, беременность).

**Примеры**
Для витамина C - 90 мг/день для взрослых; для кальция - 1000 мг/день. Превышение СДДП может быть вредным (гипервитаминоз), а недостаток - привести к дефициту.
